Información del Servidor Web

1. Instalación del Servidor Web

Para instalar el servidor web Apache en una instancia de Amazon Linux, se siguieron los siguientes pasos:

  1. Actualización del sistema: 
    sudo yum update -y
  2. Instalación de Apache: 
    sudo yum install httpd -y
  3. Inicio del servicio de Apache: 
    sudo systemctl start httpd
  4. Configuración para iniciar Apache automáticamente al arrancar el sistema: 
    sudo systemctl enable httpd
  5. Cambio del puerto de escucha de Apache a 8088: 
    Editando el archivo /etc/httpd/conf/httpd.conf y cambiando la línea Listen 80 a Listen 8088
  6. Reinicio de Apache para aplicar los cambios: 
    sudo systemctl restart httpd
Inconvenientes encontrados: Ninguno significativo, solo se requirió abrir el puerto 8088 en las reglas de seguridad de AWS.
